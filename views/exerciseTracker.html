Exercise tracker

Create a New User

POST /api/users

Add exercises

POST /api/users/:_id/exercises

GET array of all users: [project url]/api/users

Or, GET user's exercise log: GET /api/users/:_id/logs?[&from][&to][&limit]

[ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

For example:
[project_url]/api/users/60e80ccea7658997b4671aa7/logs?from=2010-12-01&to=2021-07-10&limit=5
